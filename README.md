# עו״ד גלית אביטל — אתר תדמית

אתר נחיתה סטטי לעורכת דין גלית אביטל, עם מיקוד בליווי עסקאות נדל״ן באזור ערד, הנגב, מיתר, דימונה, באר שבע והסביבה.

## סטטוס

- גרסה ראשונית: עמוד נחיתה אלגנטי, RTL, רספונסיבי.
- תמונות: כרגע איורי SVG/CSS מקוריים כדי לא להסתמך על תמונות אינסטגרם ללא אישור/גישה. ניתן להחליף בהמשך בתמונות מאושרות.
- פרטי קשר: CTA זמני `hello@example.com` עד לקבלת טלפון/מייל/ווטסאפ/אינסטגרם רשמיים.
